## 简要评价

该仓库是目前 GitHub 上最全面的 AI 系统提示词收集项目，覆盖 Anthropic、OpenAI、Google、xAI 等主要厂商的最新及历史版本，对 AI 安全研究和 Prompt Engineering 有较高参考价值。仓库创建仅 10 个月即获近 3.5 万 Star，增长极快，但高度依赖单一维护者（>92% 提交），且未声明任何开源许可证，内容使用存在法律灰色地带。Issue #1 "Source?" 的 12 条讨论也反映了社区对内容来源合法性的持续关注。
