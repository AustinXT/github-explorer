# 源代码目录 (src)

项目的源代码目录。

## 通用目录骨架

```
src/
├── app/              # 应用入口 / 路由
├── components/       # UI 组件
├── lib/              # 工具函数与配置
├── services/         # 业务逻辑层
├── hooks/            # 自定义 Hooks（前端项目）
├── types/            # 类型定义
└── constants/        # 常量
```

## 使用建议

- 根据实际技术栈调整目录结构
- 按功能模块分组，保持单一职责
- 敏感操作在服务端执行，验证用户输入
