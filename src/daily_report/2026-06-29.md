### 2. [Unclecheng-li/VulnClaw](https://github.com/Unclecheng-li/VulnClaw) · ⭐ 1,029 · Python · 新进榜

把 AI Agent + MCP 工具链 + 渗透 Skill 编排在一起，**自然语言输入 → 自动完成「信息收集 → 漏洞发现 → 漏洞利用 → 报告生成」全流程**。

要点：Agent 落地渗透测试是安全圈的高价值方向；MCP 协议让 LLM 可直接调用安全工具链，比传统 PoC 框架快一个量级。注意：仓库仅 1K star，新进榜，质量待社区验证。
