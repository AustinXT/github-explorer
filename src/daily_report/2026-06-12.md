### 1. [LMCache/LMCache](https://github.com/LMCache/LMCache) · ⭐ 8,530 · Python · 新进榜 #9

为 LLM 推理装上分布式 KV Cache 共享层，让多请求/多实例之间复用已计算好的注意力键值，显著降延迟与 GPU 占用。

要点：KV Cache 是 LLM 推理最大的隐性成本——同一段前缀在并发请求里被反复重算。LMCache 把 Cache 抽成独立层、支持跨 GPU/跨节点共享，是 vLLM、TensorRT-LLM 之外的「中间件式」补完方案；本期唯一未深度分析过的 AI 新星。
