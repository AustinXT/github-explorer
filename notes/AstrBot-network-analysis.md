## 官方文档洞察

### 官网 (astrbot.app) 价值主张

**核心定位**：Open Source Agentic AI Infrastructure — 不仅仅是聊天机器人，而是具备完整 Agent 能力的运行时。

**目标用户**：
1. 个人用户——打造 AI 伴侣/助手
2. 开发者——快速构建生产级 AI 应用
3. 团队——企业知识库、智能客服

**关键差异化**：
1. **Agentic 而非 Chatbot**：支持 Sub-Agents、复杂工作流、工具调用、上下文管理，强调「real actions」
2. **IM 原生**：直接运行在聊天 App 中，而非独立 Web 界面，强调工作流融入日常通讯
3. **情感陪伴能力**：分段回复、主动对话、人格编辑、长期记忆，这是其他框架少有的
4. **1000+ 插件生态**：国内最大规模的 AI 机器人插件市场
5. **知识库系统**：内置混合检索（dense + BM25），集成 Microsoft Markitdown 解析

**设计理念**：
- 「陪伴与能力从不对立」——既要懂情感、能陪伴，又要可靠地完成任务
- 「私は、高性能ですから!」（我可是高性能的！）——源自《Angel Beats!》的日文台词，透露二次元文化基因
