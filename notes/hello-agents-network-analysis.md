## 官方文档洞察
- 价值主张: "从零开始构建智能体"——定位为 AI Native Agent 的系统性学习教程，强调理论与实战并重
- 目标用户: 有 Python 基础的 AI 开发者、软件工程师、在校学生、自学者
- 差异化叙事: 明确区分"软件工程类 Agent"（Dify/Coze/n8n，流程驱动）和"AI 原生 Agent"（AI 驱动），教程聚焦后者；强调"穿透框架表象"，从"使用者"蜕变为"构建者"
- 内容体系: 5 大部分 16 章完整课程体系，从智能体概念到毕业设计的全流程覆盖：
  1. 基础理论（1-3章）：定义、历史、LLM 基础
  2. 动手构建（4-7章）：经典范式、低代码、框架、自研框架
  3. 高级扩展（8-12章）：记忆检索、上下文工程、通信协议、Agentic RL、评估
  4. 综合案例（13-15章）：旅行助手、Deep Research、赛博小镇
  5. 毕业设计（16章）
- 在线阅读: 基于 Docsify 构建，支持中英双语、暗黑模式、Giscus 评论系统
- Cookbook: https://book.heterocat.com.cn/
- PDF 版本: 免费开源，V1.0.2（2026-02-10），添加 Datawhale 水印防止营销号贩卖
- 下一步规划: 视频课程、HelloAgents 框架 Dev 分支维护、3W Star 后推出续作《从零开始训练智能体》
