## 生态网络

### PlayCanvas 生态矩阵

PlayCanvas 不仅是一个引擎，而是一个完整的 Web 3D 平台生态：

- **engine** (14,560★) - 核心渲染引擎（本仓库）
- **editor** (986★) - 浏览器端可视化编辑器
- **editor-mcp-server** (99★) - AI MCP 集成服务
- **awesome-playcanvas** (392★) - 社区资源精选列表
- **developer-site** (87★) - 开发者文档站
- **create-playcanvas** (3★) - CLI 项目脚手架

### 技术栈定位

PlayCanvas 占据了 Web 3D 领域中"完整游戏引擎 + 云端编辑器"的独特定位：
- 基于 WebGL2 + WebGPU 双后端
- 支持 WebXR (VR/AR)
- 内建 3D Gaussian Splatting 支持
- glTF 2.0 原生支持 + Draco 压缩
- Entity-Component-System 架构

## 官方文档洞察

### 官网价值主张 (playcanvas.com)
- **核心定位**: "Web Graphics Creation Platform" — 浏览器端 3D 图形创作平台
- **价值主张**: 协作式构建 HTML5 游戏和可视化应用，无需安装
- **差异化叙事**:
  - **WebGPU Ready**: 首批支持 WebGPU 的生产级 3D 引擎，含计算着色器，同时保持 WebGL2 兼容
  - **3D Gaussian Splatting**: 业界领先的 3DGS 支持，内建编辑和优化工具
  - **多种开发方式**: 可视化编辑器 / npm 引擎集成 / React 组件 / Web Components
- **目标用户**: 游戏工作室、互动广告、产品配置器、AR/VR 体验、赌场游戏、建筑可视化
- **设计哲学**: "Build It Your Way" — 多路径灵活开发，不强制单一方法论
- **商业模式**: 免费层（无限公开项目/1GB 存储/免费托管） + 付费高级计划

## 竞品清单

| 竞品 | Stars | Forks | 定位 | 与 PlayCanvas 对比 |
|------|-------|-------|------|-------------------|
| **Three.js** (mrdoob/three.js) | 111,483 | 36,303 | 通用 WebGL 3D 库 | 更底层，非游戏引擎；生态最大；无内建编辑器 |
| **Babylon.js** (BabylonJS/Babylon.js) | 25,233 | 3,619 | 全功能 Web 3D 引擎 | 最直接竞品；微软支持；功能集更大；社区更活跃 |
| **PixiJS** (pixijs/pixijs) | 46,795 | 4,987 | 2D WebGL 渲染器 | 2D 领域；非直接竞品但生态有重叠 |
| **Phaser** (phaserjs/phaser) | 39,221 | 7,124 | 2D HTML5 游戏框架 | 2D 游戏框架；非 3D 竞品 |
| **A-Frame** (aframevr/aframe) | 17,501 | 4,343 | WebXR 框架 (基于 Three.js) | VR/AR 侧重；声明式 HTML 开发方式 |

**竞品格局分析**:
- PlayCanvas 在纯 Web 3D 游戏引擎赛道中，最直接的竞品是 **Babylon.js**（Stars 约 1.7x）
- Three.js 虽 Stars 最高但定位为渲染库而非完整引擎
- PlayCanvas 的独特优势：**云端协作编辑器 + MIT 开源引擎** 的组合在竞品中独一无二
- Babylon.js 背后有微软资源；PlayCanvas 是独立公司运营

## 快速判断

- **是否值得深入**: 是
- **初步定位**: Web 3D 领域最成熟的开源游戏引擎之一，公司驱动 + MIT 开源。在"完整游戏引擎 + 云端编辑器"组合中定位独特，适用于浏览器端 3D 游戏、互动广告、产品配置器、XR 体验等场景。WebGPU 先发优势和 3D Gaussian Splatting 支持是差异化亮点。
- **作者可信度**: 高。PlayCanvas 是一家成立于 2011 年的伦敦公司（Will Eastcott & David Evans 联合创始），拥有稳定的核心工程团队（5-6人持续高频贡献），12年持续开发历史，服务过 Disney/BMW/Samsung/Snap 等大客户。71个公开仓库构成完整生态。
- **竞品格局**: 中等竞争。最直接竞品 Babylon.js（微软背景/25K★）在社区规模和功能丰富度上略胜；Three.js（111K★）定位不同但生态压倒性；PlayCanvas 的核心护城河在于"引擎 + 云端编辑器"的一体化体验和商业化成熟度，是 Web 3D 赛道中少数成功商业化运营的开源项目。
