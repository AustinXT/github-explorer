## 10. 综合评估

### 项目画像

social-analyzer 是一个 **OSINT 社交媒体用户名侦查工具**，定位于帮助安全研究人员和执法人员在 1000+ 社交媒体网站上查找和分析目标用户的画像。它在同类工具中以**功能最全面**著称（API+CLI+Web+评分+OCR+元数据），但在**社区活跃度和维护频率上明显落后**于 Sherlock 和 Maigret。

### SWOT 分析

| 维度 | 内容 |
|------|------|
| **优势 (S)** | 功能全面（三种接口+评分+OCR+可视化）；22K Star 的品牌影响力；被执法机构实际使用；支持 Python 和 Node.js 双栈 |
| **劣势 (W)** | 维护频率极低（数月一次）；社区贡献者极少（Bus Factor=1）；AGPLv3 限制商业使用；npm 下载量极低（129/月） |
| **机会 (O)** | OSINT 领域持续增长；可与 AI 工具链集成；执法/企业安全市场需求稳定 |
| **威胁 (T)** | Sherlock（74K Star）和 Maigret（3000+ 网站覆盖）持续蚕食市场；项目若长期不维护可能被社区遗弃；社交平台反爬策略持续升级可能导致检测失效 |

### 关键风险

1. **维护风险：** 项目基本由一人维护，最近提交间隔数月，存在被废弃的风险
2. **技术债务：** 15.4MB 的 JavaScript 代码量表明可能包含大量内联的网站检测数据（sites.json），维护成本高
3. **许可证风险：** AGPLv3 对商业集成有较强限制
4. **准确性衰减：** 社交平台频繁更新界面和 API，长期不维护将导致检测准确率持续下降
