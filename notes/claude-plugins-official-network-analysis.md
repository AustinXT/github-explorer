## 竞品清单

| 竞品 | 定位 | 对比 |
|------|------|------|
| **Cursor Extensions** | Cursor IDE 的扩展生态 | IDE 级集成，GUI 优先；claude-plugins-official 是 CLI 优先 |
| **GitHub Copilot Extensions** | Copilot 的扩展市场 | 依托 GitHub 生态，用户基数大；但插件灵活性不如 Claude 插件系统 |
| **Aider** | 开源 AI 编程工具 | BYOK 模式，成本低；但无官方插件生态 |
| **Codex (OpenAI)** | OpenAI 的编码 Agent | 注重自主性；插件系统不如 Claude 成熟 |
| **Windsurf (Cognition)** | AI IDE（前身 Devin） | 全自动 Agent 定位；与 Claude Code 插件系统走不同路线 |
| **ComposioHQ/awesome-claude-plugins** | Claude 插件精选列表 | 非竞品，而是互补的社区精选 |

**差异化优势**：claude-plugins-official 是唯一一个由 AI 模型厂商直接维护的官方插件目录，与 Claude Code CLI 和 VS Code 扩展深度集成，支持"一次安装，两处可用"。

## 项目展示素材

### README 核心信息

- **定位**：A curated directory of high-quality plugins for Claude Code
- **结构**：`/plugins`（内部）+ `/external_plugins`（第三方）
- **安装命令**：`/plugin install {plugin-name}@claude-plugins-official`
- **浏览命令**：`/plugin > Discover`
- **插件标准结构**：plugin.json (必须) + .mcp.json (可选) + commands/ + agents/ + skills/ + README.md
