## 官方文档洞察

### README 特点

- 极其详尽（500+ 行），涵盖安装、配置、使用、部署、STT、API 模式、远程服务器等全流程
- 提供 Demo 视频展示完整工作流
- 8 种语言的 README 版本，全球化意识强
- 明确标注「Active Work in Progress」和「zero roadmap and zero funding」— 管理预期诚实
- 配置说明详细，有 FAQ 和 Troubleshooting

### 核心架构（来自 DeepWiki）

六层架构：
1. **User Layer**：Web 界面（React, port 3000）+ CLI
2. **Orchestration Layer**：会话管理、智能路由
3. **Agent Layer**：5 个专业 Agent — CasualAgent、CoderAgent、FileAgent、BrowserAgent、PlannerAgent
4. **Capability Layer**：LLM 抽象、Web 自动化、Memory、代码执行
5. **External Services**：本地 LLM、云 API、Docker 化基础设施
6. **Configuration**：config.ini + .env

### AgentRouter（智能路由）

多阶段分类流程：
1. 语言检测与翻译
2. 复杂度评估（高复杂度自动分配 PlannerAgent）
3. BART 零样本分类 + LLM 分类并行执行
4. 投票机制选择最优 Agent
