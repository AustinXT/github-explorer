### 竞争优势分析
- **vs Emdash**：Superset 先发优势（Star 数 2.4x），产品更成熟（已到 v1.4.7），桌面原生应用体验更好
- **vs Cursor/Windsurf**：不是替代关系，是互补层——Superset 编排多个 agent，Cursor 是单 agent 深度 IDE
- **vs tmux 方案**：Superset 提供图形界面、内置 diff 审查、IDE 集成，易用性远超终端方案
- **核心差异化**：唯一一个同时具备「桌面原生 + Agent 无关 + Worktree 隔离 + 内置代码审查」的产品

## 项目展示素材

### README 核心卖点

1. **「Code 10x Faster With No Switching Cost」** — 主 tagline
2. **并行执行**：同时运行 10+ coding agent
3. **Worktree 隔离**：每个任务独立分支和工作目录
4. **Agent 监控**：统一监控面板 + 完成通知
5. **内置 Diff 审查**：不离开应用即可检查和编辑 agent 修改
6. **Workspace 预设**：自动化环境搭建
7. **IDE 集成**：一键在常用编辑器中打开

### 技术架构亮点
- Monorepo 结构（6 个主应用 + 多个共享包）
- Electron 桌面应用为旗舰产品
- 配套 web、API、admin 应用提供后端服务
- 用 Bun + Turborepo 管理构建

## 快速判断

**一句话定位**：AI Agent 时代的「编排层」——不做 Agent 本身，而是让开发者在一台机器上并行运行、监控、审查多个 AI coding agent 的桌面工具。

**核心价值**：解决了 2026 年 AI coding agent 爆发后的「多 Agent 协调」痛点——当开发者同时使用 Claude Code、Codex、Gemini CLI 等多个 agent 时，需要隔离环境、统一监控和快速审查。

**团队评估**：⭐⭐⭐⭐⭐
- 3 位前 YC CTO 创始人，均有大厂（Amazon/Google/Facebook）经验
- YC S26 入选，获得顶级加速器背书
- 团队执行力极强：5 个月内从 0 到 8.6K Star，3 月单月 26 个版本发布

**增长评估**：⭐⭐⭐⭐⭐
- 典型的 Product-Market Fit 信号：3 月 Star 爆发式增长（+5,957，占总量 69%）
- Product Hunt 日榜 #1，HN 多次曝光
- 声称被 Microsoft、OpenAI、Netflix 等团队使用

**风险点**：
1. **许可证风险**：ELv2 不是传统开源许可证，可能影响企业采纳和社区贡献意愿
2. **社区参与度偏低**：贡献几乎完全由 3 位创始人完成，社区 PR/Issue 参与度低
3. **赛道拥挤**：2026 年「并行 AI Agent 编排」赛道快速涌入大量竞品（Emdash、Uzi、Genie 等）
4. **平台依赖**：强依赖 Electron，Windows/Linux 尚标注「未测试」
5. **商业模式不明确**：当前免费，未公布付费计划

- 技术亮点：git worktree 作为隔离方案的优雅设计
