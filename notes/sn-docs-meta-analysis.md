### 1.2 依赖分析

**后端（Python / Django）** — 42 个生产依赖，16 个开发依赖：
- Web 框架：Django 5.x + Django REST Framework
- 数据库：PostgreSQL（psycopg3）
- 缓存/队列：Redis + Celery
- 存储：boto3 + django-storages（S3）
- AI 集成：OpenAI SDK + pydantic-ai + langfuse（可观测性）
- 认证：mozilla-django-oidc（OIDC SSO）
- 协作编辑：pycrdt（CRDT 实现）
- 其他：sentry-sdk、gunicorn、uvicorn

**前端（TypeScript/React）** — 独立 monorepo（yarn workspace），主要包：
- `apps/impress`：主应用（React/Next.js）
- `apps/e2e`：端到端测试
- `servers/y-provider`：Yjs 协作服务端
- `packages/i18n`：国际化

**邮件模块** — 2 个依赖：mjml + @html-to/text-cli（邮件模板）

**基础设施**：Dockerfile（后端 + 前端）、Makefile、Helm Charts
