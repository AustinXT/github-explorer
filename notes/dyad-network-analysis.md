## 5. 官方文档洞察

### 官网 (dyad.sh)

**核心定位**：「灵活、本地、开源的 AI 应用构建器」

**产品哲学**：
> "你应该感到像是所有者，而不是租户。" — Will Chen

**核心卖点**：
1. **零锁定**：代码可自由导入/导出，不绑定任何平台
2. **本地优先**：在用户机器上运行，隐私有保障
3. **模型自由**：支持任意 AI 模型（自带 API Key 或使用免费模型）
4. **全栈能力**：支持身份验证、数据库、后端功能、一键部署

**定价模式**：

| 方案 | 价格 | 核心特性 |
|------|------|----------|
| Free | $0 | 本地构建、无限应用、BYOK、5 条/天免费 Agent 消息 |
| Pro | $20/月 | 200 AI 额度/月、Pro 模式 |
| Max | $79/月 | 900 AI 额度/月、优先支持 |

**支持平台**：macOS、Windows（Linux beta）

### 文档站
- 官方文档：https://www.dyad.sh/docs/
- 包含：快速入门、集成指南（GitHub/Supabase/Vercel）、发布说明、模板指南、竞品对比页

## 6. 竞品清单

| 竞品 | 类型 | 定价 | 与 Dyad 的关键差异 |
|------|------|------|---------------------|
| **Lovable** | 云端 AI 应用构建器 | $25/月起 | 云端运行、闭源、有供应商锁定 |
| **v0 (Vercel)** | 云端 UI 生成器 | $20/月 | 仅生成 React/Tailwind UI 组件，功能范围窄 |
| **Bolt.new** | 浏览器端 IDE | $20/月 | 基于 WebContainers，不在本地运行 |
| **Bolt.diy** | 开源自托管 | 仅 API 成本 | 需要命令行安装，门槛更高 |
| **Replit** | 云端 IDE | $20-25/月 | 重度使用者月费可达 $100-300 |
| **Cursor** | AI 代码编辑器 | $20/月 | 面向专业开发者，非应用构建器 |
| **Windsurf** | AI 代码编辑器 | $15/月 | 定位类似 Cursor |

**Dyad 的差异化优势**：
1. 唯一同时满足「本地运行 + 开源 + 全栈构建」的产品
2. 免费层真正可用（Ollama 本地模型 + 免费 OpenRouter 模型）
3. 代码完全归用户所有，无平台依赖

## 10. 快速判断

### 项目定性

| 维度 | 评估 | 说明 |
|------|------|------|
| **活跃度** | ★★★★★ | 每周发版，2 天前仍有提交，89 个 release |
| **增长性** | ★★★★★ | 11 个月 20K stars，爆发式增长 |
| **社区参与** | ★★★☆☆ | 3000+ Reddit 成员，但代码贡献高度集中于创始人（84%） |
| **商业化** | ★★★★☆ | 有明确 Pro/Max 付费方案，Dyad Tech, Inc. 实体运营 |
| **技术质量** | ★★★★☆ | Electron + React + TypeScript 全栈，架构清晰，集成丰富 |
| **生态成熟度** | ★★★☆☆ | 尚处早期，周边工具仓库 star 均 < 30 |
| **可持续性风险** | 中等 | 极度依赖单一创始人，Bus Factor = 1 |

### 一句话总结
Dyad 是 2025-2026 年 AI 应用构建赛道增长最快的开源项目之一，由前 Google 工程师 Will Chen 独立打造，凭借「本地运行 + 开源 + 零锁定」的差异化定位在 v0/Lovable/Bolt 等云端竞品中杀出重围，但项目高度依赖单一创始人，社区代码贡献生态仍需培育。

### 值得关注的信号
1. **Vibe Coding 趋势的受益者**：Dyad 精准卡位"非专业开发者用 AI 构建应用"赛道
2. **Google 背景加持**：创始人的 Mesop 项目经验直接转化为产品力
3. **Bus Factor 风险**：84% 提交来自一人，需观察团队扩张节奏
4. **商业模式可行性**：Free + Pro 分层合理，但免费层足够强大可能影响转化
5. **FSL 许可证策略**：Pro 功能采用 FSL 1.1 许可，平衡开源与商业化
