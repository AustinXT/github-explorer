## 9. 项目展示素材

### 一句话介绍
> A free, open source, and extensible speech-to-text application that works completely offline.

### 核心卖点
- **按住说话**：按下快捷键说话，松开自动转文字粘贴
- **完全离线**：音频不出本机，隐私至上
- **跨平台**：macOS (Intel/Apple Silicon)、Windows、Linux
- **多模型**：Whisper (GPU加速) + Parakeet V3 (CPU优化) + Moonshine + SenseVoice
- **可扩展**：MIT 许可，"最可 Fork 的语音转文字应用"

### 技术亮点
- Tauri 2.9 (Rust + React/TypeScript) 架构
- 管理器模式：AudioRecording / Model / Transcription / History
- 类型安全 IPC (tauri-specta)
- GPU 加速：Metal / CUDA / Vulkan
- 11 种语言国际化支持

### 创作故事
CJ Pais 因手指骨折打石膏，单手打字困难，于是创建了 Handy。项目从个人需求出发，发展为 18K+ Stars 的热门开源项目。

### 总体判断

**强烈推荐关注。** Handy 是 2025-2026 年语音转文字领域最亮眼的开源项目之一。

**核心优势**：
1. 精准的产品定位 - 填补了免费、开源、离线、跨平台语音转文字的空白
2. 真实的创作动机 - 源自创始人的实际需求，产品方向聚焦
3. 爆发式增长 - 9 个月 18K Stars，增速仍在加快
4. 极高的发布频率 - 53 个版本，展现强大的执行力
5. 差异化明确 - "不做最好的，做最可 Fork 的"

**潜在风险**：
1. **创始人依赖**：83% 提交来自 cjpais 一人，总线因子 (bus factor) 低
2. **跨平台兼容性**：Linux/Wayland、AMD GPU 等兼容性问题仍是挑战
3. **商业可持续性**：完全免费模式下，长期维护动力取决于赞助和社区贡献
4. **功能边界**：作为 "一个工具一个功能" 的定位，需平衡扩展需求和简洁性

**适合场景**：
- 需要隐私优先的离线语音转文字
- 不愿为听写工具付费的用户
- Linux 桌面用户（大多数竞品不支持 Linux）
- 希望定制/Fork 语音转文字工具的开发者
