## 竞品分析
直接竞品（本地 LLM 桌面客户端）：
1. **LM Studio** — 最流行的竞品，闭源，GUI 友好，商业化
2. **Ollama** — CLI 优先，API 导向，"Docker for LLMs"
3. **GPT4All** — Nomic AI 出品，轻量级，广泛硬件兼容
4. **AnythingLLM** — 文档聊天 + Agent，团队协作功能
5. **Open WebUI** — 自托管 Web UI，集成 Ollama

Jan 的差异化定位：
- 隐私优先 + 完全离线
- 集成云端 API（混合模式）
- OpenAI 兼容 API 服务器
- MCP 协议支持（agentic 能力）
- 跨平台：桌面（Windows/Mac/Linux）+ 移动端开发中
