### 关键洞察

1. **爆发式增长期：** 2023 年起提交量从年均 400 暴增至 3,900+（2025），项目经历了从小团队项目到商业化产品的转变，与 Kestra 获得融资、团队扩张的时间线吻合。

2. **v1.0 里程碑：** 经过 5.5 年孵化期后，于 2025-09 发布 1.0 GA，标志产品进入生产就绪阶段。此后以 ~2 个月为周期发布 minor 版本，节奏稳健。

3. **全栈架构：** Java 后端 + Vue 前端的全栈项目，前端修改频率甚至略高于后端，说明 UI/UX 是产品差异化的重要战场。

4. **企业级治理：** 多版本并行维护、高密度补丁发布、dependabot 自动化依赖管理，均体现商业软件的工程治理水平。

5. **欧洲基因：** 核心团队集中在法国/西欧，工作日+标准工时模式，周末几乎无活动，是典型的欧洲商业公司开发文化。
