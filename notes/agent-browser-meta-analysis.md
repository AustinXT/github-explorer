### 关键洞察

1. **超高速迭代**：71 天 60 个版本、452 次提交，属于极高强度的开发节奏，反映了 Vercel Labs 对此项目的重视和投入。
2. **Bug 修复驱动**：fix 类型占比 44% 说明项目在功能快速推进的同时面临较多稳定性挑战，这在浏览器自动化类工具中是常见的（DOM 交互复杂性高）。
3. **单核开发模式**：Chris Tate 贡献了 58% 的代码，项目的巴士因子（Bus Factor）为 1，存在人员风险。
4. **Rust 为核心**：CLI 的 Rust 代码（3.2 万行）是主要开发热点，TypeScript SDK 较轻量，说明性能敏感的浏览器控制逻辑主要在原生层实现。
5. **文档投入可观**：docs/src 变更 282 次，MDX 文件 24 个，说明项目重视开发者体验和文档建设。
