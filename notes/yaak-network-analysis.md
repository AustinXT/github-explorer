### 技术栈

**前端**：
- React + TypeScript（UI 层）
- Vite+（构建工具链，最近从 Vite 迁移）
- CSS（自定义样式）
- CodeMirror（编辑器组件）

**后端/核心**：
- Rust（核心引擎：HTTP/gRPC/SSE/WebSocket 客户端）
- Tauri 2.x（桌面框架，取代 Electron）
- SQLite（本地数据存储）
- tokio（异步运行时）

**关键 Rust crate**：
- `reqwest`（HTTP 客户端）
- `rustls`（TLS）
- `keyring`（操作系统钥匙链集成）
- `tauri` 2.9.5（桌面运行时）
- `serde`/`serde_json`（序列化）
- `ts-rs`（TypeScript 类型生成）

**插件系统**：
- Node.js sidecar（插件运行时）
- gRPC 通信（app <-> 插件）
- 支持认证插件、模板标签插件、UI 插件

**CI/CD**：
- GitHub Actions（ci.yml, release-app.yml, release-api-npm.yml, release-cli-npm.yml）
- Flathub 自动发布
- npm 包发布（@yaakapp/api, CLI）
- Claude Code 集成（claude.yml - AI 辅助 code review）

**分发渠道**：
- GitHub Releases（.dmg, .exe, .deb, .rpm, .AppImage）
- npm（@yaakapp/api, CLI）
- Flathub（Linux flatpak）

Yaak 是 Insomnia 创始人 Gregory Schier 用 Tauri + Rust 重新打造的隐私优先桌面 API 客户端，凭借创始人的行业声誉和扎实的工程能力在 18 个月内积累 18K+ Stars，但项目高度依赖单人开发、社区贡献门槛严格、Issues 已外迁至独立平台，整体呈现出「优秀的个人作品」而非「社区驱动的开源项目」的特征。

6. **日历版本号策略**：采用 `vYYYY.M.patch` 版本号，暗示长期维护承诺
