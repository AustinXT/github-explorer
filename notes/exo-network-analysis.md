### 技术研究

- **SPARTA**（Sparse Parameter Averaging for Distributed Training）：EXO Labs 提出的分布式训练方法，可实现 1,000x 通信量缩减。发表于官方博客 [Day 12](https://blog.exolabs.net/day-12/)。
- **流水线并行推理**：将 LLM 切分为多个 shard（连续层），分配到不同设备。单请求时多设备有开销，但并发请求时接近线性扩展（3 台设备处理 2.2x 吞吐量）。
- **张量并行 + RDMA**：Thunderbolt 5 RDMA 支持下实现设备间超低延迟通信，4 台设备 3.2x 加速。

## 项目展示素材

### 口号
> **"exo connects all your devices into an AI cluster."**
> "Run frontier AI locally."

### 核心卖点
1. **设备自动发现**：运行 exo 的设备自动互相发现，无需手动配置
2. **RDMA over Thunderbolt 5**：设备间延迟降低 99%
3. **拓扑感知自动并行**：基于实时设备拓扑自动选择最优模型拆分方案
4. **张量并行**：2 台设备 1.8x 加速，4 台设备 3.2x 加速
5. **MLX 深度集成**：Apple Silicon 原生推理
6. **多 API 兼容**：OpenAI / Claude / Ollama API 全兼容
7. **内置仪表盘**：可视化集群管理与模型交互

### 典型使用场景
- 4 台 Mac Mini M4 + MacBook Pro M4 Max 组成 ~$5,000 的 AI 集群，相比单张 H100 GPU 大幅降低成本
- 4 x 512GB M3 Ultra Mac Studio 运行 DeepSeek v3.1（671B 参数，8-bit）
- 两台 NVIDIA DGX Spark + M3 Ultra Mac Studio 异构集群，实现 2.8x 性能提升
