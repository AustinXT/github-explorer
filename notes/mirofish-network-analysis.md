## 生态网络

### BettaFish 与 MiroFish 的关系

两个项目构成上下游**姊妹产品**：
- **BettaFish**（微舆，39,647 Stars）：多Agent舆情分析助手，负责从现实世界采集和分析舆情数据
- **MiroFish**（38,210 Stars）：群体智能预测引擎，在 BettaFish 分析结果的基础上进行未来推演
- **MindSpider**（336 Stars）：专为舆情分析设计的 AI 爬虫，提供数据采集能力

三者形成"采集 → 分析 → 预测"的闭环。README 中演示视频明确展示了使用 BettaFish 生成的舆情报告作为 MiroFish 的输入材料。

## 快速判断

**一句话定位**：MiroFish 是目前开源社区中最受关注的群体智能预测引擎，通过构建数千 AI 智能体的平行数字世界来模拟预测未来，4 个月内获 38K Stars。

**核心亮点**：
1. **赛道独特**：在"多Agent社会模拟 × 预测引擎 × 开源产品"这一交叉领域几乎无直接竞品
2. **增长惊人**：创建 4 个月即达 38K Stars，两次登顶 GitHub 全球 Trending #1
3. **商业背书**：获盛大集团陈天桥 3000 万人民币投资，已成立独立团队并招聘
4. **产品链完整**：与 BettaFish、MindSpider 构成采集→分析→预测的闭环
5. **学术基座扎实**：基于 OASIS（arxiv:2411.11581）百万Agent仿真框架

**主要风险**：
1. **单人依赖**：219 次提交中仅有 1 位核心开发者，Bus Factor = 1
2. **稳定性不足**：500/504 错误频发，Issue 中大量用户反映流程卡住
3. **Zep 依赖锁定**：核心记忆层依赖 Zep Cloud 付费服务，免费额度消耗快，社区对此有强烈不满
4. **社区治理薄弱**：健康度仅 42/100，无贡献指南/Issue模板/行为准则
5. **仍处 0.x 早期**：功能不稳定，文档不完善，不适合生产使用
6. **AGPLv3 许可证**：对商业二次开发有较强限制

**趋势判断**：MiroFish 处于**爆发式增长期**，从媒体关注度、Star 增速、投资背书来看势头很强。但项目成熟度与其热度严重不匹配——单人开发、Bug 频发、核心依赖受限。未来 3-6 个月是关键窗口期，需要快速组建团队、解决 Zep 依赖、提升稳定性，否则热度可能难以转化为持续的用户留存。
