### 依赖极简

Go 模块仅 **7 个直接依赖 + 2 个间接依赖**（go.sum 仅 20 行）：

- `github.com/tdewolff/parse/v2` — HTML/CSS 解析
- `golang.design/x/clipboard` — 剪贴板
- `golang.org/x/image`、`net`、`sys`、`text` — Go 官方扩展库

依赖数量极少，体现了「零依赖」哲学——引擎核心功能几乎全部自研。

所有发布均为 Pre-release / Nightly 状态，说明项目尚未推出正式稳定版，但已建立了成熟的 CI/CD 每日发布管线。

### CI/CD 管线

3 个 GitHub Actions 工作流：
- `ci-build-test.yml` — 多平台 CI（Windows 2022/2025，Vulkan SDK + MinGW）
- `nightly.yml` — 每日 Nightly 自动发布
- `main.yml` — 主分支工作流
