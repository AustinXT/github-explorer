**构建系统**：Rust 侧使用 Cargo workspace（7 个 crate），Python 侧使用 uv + maturin（PyO3 绑定）。项目版本 `0.0.8`，Rust edition 2024，要求 rustc >= 1.90。

**Workspace 结构**（7 个 crate）：
- `monty` — 核心解释器
- `monty-cli` — 命令行工具（默认成员）
- `monty-python` — Python 绑定（PyO3/maturin）
- `monty-js` — JavaScript/WASM 绑定
- `monty-type-checking` — 类型检查
- `monty-typeshed` — typeshed 集成
- `fuzz` — 模糊测试
