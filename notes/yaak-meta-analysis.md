**总代码行数**：约 92,500 行（TS/TSX ~55,000 + Rust ~36,000 + 其他 ~1,500），属于**中型项目**。

### 核心代码语言

Rust + TypeScript/TSX 各占约 1/3，构成典型的 **Tauri 桌面应用架构**：Rust 负责后端核心（HTTP/gRPC/模板/同步），TypeScript/React 负责前端 UI。

NPM 依赖极少（仅 13 个），说明前端高度自研。Rust 端 38 个 workspace 依赖 + 21 个内部 crate，体现了模块化设计。

**核心开发者 Gregory Schier 贡献了 95.8% 的提交**（合并同一人的两个账号：2,310 + 16 = 2,326 / 2,433 非 bot 提交）。这是一个典型的**独立开发者主导项目**。

### 协作模式判定

- **模式**：独立开发者 + 零散社区贡献
- **合并提交**：49 次（占 2.0%），说明 PR 合并极少，大部分代码直接推送到主分支
- **社区参与**：54 个社区贡献者，但每人仅 1-8 次提交，主要为小修小补（翻译、bug 修复、依赖更新）
- **Bus Factor**：1 — 如果 Gregory Schier 停止开发，项目将面临严重停滞风险
